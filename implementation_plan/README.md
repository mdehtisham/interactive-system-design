# Implementation Plan

This folder contains numbered implementation plan documents for the interactive system design learning platform. Plans are created progressively — each file covers a discrete, deployable phase of the project.

| File | Phase | Status |
|---|---|---|
| `01-project-setup.md` | Project scaffolding, tooling, CI/CD | Ready |
| `02-content-infrastructure.md` | MDX pipeline, topic routing, animation shell | Pending |

Plans are written before implementation begins. Do not start a phase without a corresponding plan file.
